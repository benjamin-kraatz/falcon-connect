# FALCON Connect Demo App 02 (Target App)

This is the second demo app for FALCON Connect. It is meant to represent and demo the target app in a FALCON Connect integration.
The dependencies are minimal and only depend on the FALCON Connect SDK, to mirror a real-world application acting as the target app as closely as possible.

For the source app demo, see [demo-01](../demo-01).

## Running Locally

Run the demo app in development with:

```bash
bun run dev
```

This will start the demo app at [http://localhost:4102](http://localhost:4102).
